# Images Directory

Place your images in this directory.

## Required Images

### Profile Photo

- **File**: `profile.jpg`
- **Size**: 800x800px (square)
- **Format**: JPG or PNG
- **Description**: Your professional headshot for the hero section and about page

## Optional Images

### Blog Images (`blog/` subdirectory)

- Place featured images for your blog posts here
- Recommended size: 1200x600px (2:1 aspect ratio)
- Name descriptively: `python-tutorial.jpg`, `3d-viz.jpg`, etc.

### Gallery Images (`gallery/` subdirectory)

- Place personal photos, conference photos, travel photos here
- Recommended size: 1200x900px (4:3 aspect ratio)
- Name with categories: `conference-icml-2024.jpg`, `travel-paris.jpg`, etc.

### Research Images (`research/` subdirectory)

- Methodology diagrams
- Results visualizations
- Charts and graphs
- Recommended size: varies, keep under 2MB each

## Image Optimization Tips

1. **Compress images** before uploading

   - Use tools like TinyPNG, ImageOptim, or Squoosh
   - Aim for < 500KB per image

2. **Use appropriate formats**

   - JPG for photos
   - PNG for graphics with transparency
   - SVG for logos and icons

3. **Responsive images**
   - Include different sizes for mobile/desktop if needed
   - Use modern formats (WebP) with fallbacks
